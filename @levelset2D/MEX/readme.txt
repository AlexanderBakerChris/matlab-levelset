
== Unix ==

To compile on unix-based systems, run 'make' in this directory.


== Windows ==

To compile on Windows, follow directions at:
http://www.mathworks.com/matlabcentral/fileexchange/loadFile.do?objectId=13321&objectType=file

Then run the 'make.m' script within MATLAB.

The strategy in section 4 has been verified to work using
MATLAB 7.5.0 (R2007b) and Visual C++ Express Edition 2008.
